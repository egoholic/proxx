# What about tests?
I had not enough time to make tests. Actually there is no real need for tests because it is a test task and there will be no future changes and regression.

You can check how I write tests in other my libs (I usually use ginkgo for BDD).

How tests could be implemented?

1. We can implememt a bot that will play the game and check if it works fine.
2. We can use the same seed for black holes randomizer and cover one or several versions of board.
